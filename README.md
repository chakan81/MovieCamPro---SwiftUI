# MovieCamPro-SwiftUI

## Detailed movie camera control interface for SwiftUI

You can configure following features manually in SwiftUI environment.

> - dimensions
> - frame rate
> - camera lens (i.e. wide, ultra wide, telephoto)
> - slo-mo
> - zoom
> - exposure
> - image stabilizer
> - HDR
> - DolbyVision
> - and so on
